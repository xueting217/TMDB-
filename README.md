# TMDB 高分电影数据爬取工具
该工具基于 Python 实现 TMDB（The Movie Database）高分电影榜单数据的爬取、清洗与结构化存储，可自动抓取指定页数的高分电影信息并导出为 UTF-8 编码的 CSV 文件，便于后续数据分析与使用。

## 功能特性
- **多页数据爬取**：自动爬取 TMDB 高分电影榜单前 5 页的电影列表；
- **详情信息提取**：获取每部电影的名称、年份、上映时间、类型、时长、评分、语言、导演、作者、宣传语、简介等核心信息；
- **数据清洗标准化**：对爬取的原始数据进行格式化处理（如时长统一转分钟、年份去括号、上映时间提取标准格式等）；
- **结构化存储**：将清洗后的电影数据保存为 CSV 文件，适配 Excel、Python 数据分析库等工具直接使用。

## 代码核心逻辑说明
### 1. 常量定义
定义 TMDB 网站基础地址、榜单页面地址及 CSV 文件存储路径，便于后续维护修改。
```python
TMDB_BASE_URL = "https://www.themoviedb.org"  # 网站基础URL
TMDB_TOP_URL = "https://www.themoviedb.org/movie/top-rated"  # 榜单第一页
TMDB_TOP_URL_2 = "https://www.themoviedb.org/discover/movie/items"  # 榜单第2-5页
MOVIE_LIST_FILE = "csv_data/movie_list.csv"  # 结果文件路径
```

### 2. 数据清洗函数
对爬取的原始数据进行标准化处理，确保数据格式统一：
| 函数名                  | 功能说明                                  |
|-------------------------|-------------------------------------------|
| `get_movie_year`        | 去除年份中的括号，返回纯数字年份          |
| `get_movie_publish_date`| 从原始文本中提取 `YYYY-MM-DD` 格式上映时间 |
| `get_movie_cost_time`   | 将“Xh Xm”格式时长转换为总分钟数（数值型） |

### 3. 核心业务函数
| 函数名              | 功能说明                                  |
|---------------------|-------------------------------------------|
| `get_movie_info`    | 根据电影详情页 URL 爬取并解析单部电影信息 |
| `save_all_movies`   | 将所有电影信息写入 CSV 文件               |
| `main`              | 主函数：遍历榜单页面 → 爬取电影列表 → 获取详情 → 保存数据 |

## 数据字段说明
CSV 文件包含以下字段
| 字段名   | 格式/说明                                  |
|----------|-------------------------------------------|
| 电影名字 | 字符串，去除首尾空格                      |
| 年份     | 纯数字字符串（如 2023），无括号           |
| 上映时间 | `YYYY-MM-DD` 标准日期格式                 |
| 类型     | 多类型用逗号分隔（如 剧情,悬疑）          |
| 时长     | 整数，单位为分钟（如 150）                |
| 评分     | 字符串，保留原始评分精度                  |
| 语言     | 字符串，去除首尾空格                      |
| 导演     | 多导演用逗号分隔                          |
| 作者     | 多作者用逗号分隔                          |
| 宣传语   | 字符串，去除首尾空格                      |
| 简介     | 字符串，去除首尾空格                      |

## 注意事项
**页面结构变更**：TMDB 网站页面结构若更新，需同步调整 XPath 路径（如 `movie_document.xpath` 中的路径）
